## 🖥️ Preview
 
> Dark pixel-art aesthetic inspired by retro gaming UI — featuring a city skyline background, scanline overlay, animated terminal, and playing-card skill deck.
 
---
 
## ✨ Features
 
- 🎨 **Pixel-art design** — Press Start 2P font, dashed borders, orange + cyan accents
- 🌆 **Animated city background** — pixel skyline on every section
- ⌨️ **Typing animation** — rotating role titles in the hero section
- 💻 **Terminal window** — shows whoami, skills, and projects like a real shell
- 🃏 **Skill deck** — skills displayed as playing cards (K / Q / J / JOKER)
- 📊 **Quick Stats card** — CGPA, GitHub repos, LeetCode count, achievements
- 🏆 **Achievement cards** — LEGENDARY / EPIC / RARE / COMMON rarity badges with XP bars
- 🕹️ **Level timeline** — experience section with LEVEL boxes and dotted connectors
- 📱 **Responsive** — works on desktop and mobile
- ⚡ **Zero dependencies** — pure HTML, CSS, and JavaScript

## 📦 How to Run Locally
 
```bash
# 1. Clone the repository
git clone https://github.com/CodeIt-Abhay/CodeIt-Abhay.github.io.git
 
# 2. Open in browser
cd CodeIt-Abhay.github.io
open index.html
# or just double-click index.html
```
 
No build step. No npm install. Just open and go. ✅

## 🌐 Deployment
 
This site is deployed via **GitHub Pages** and is live at:
 
**[https://CodeIt-Abhay.github.io](https://CodeIt-Abhay.github.io)**
 
To deploy your own fork:
1. Fork this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Save — your site will be live in ~2 minutes
